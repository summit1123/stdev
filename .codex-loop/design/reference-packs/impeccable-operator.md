# Reference Pack: impeccable-operator

Inspired by `impeccable`, use this when the product is a dashboard, control surface, review console, or operational tool that must feel sharp without looking noisy.

## Core feel

- precise, confident, and quiet
- visual rhythm comes from spacing, alignment, type scale, and state contrast
- strong operational hierarchy without dashboard confetti
- polished enough to ship, restrained enough to trust

## Structural cues

- prefer split panes, inspectors, logs, tables, and purposeful summaries
- keep one dominant task path on each screen
- use tinted neutrals and restrained accents instead of loud gradients
- status colors should encode meaning, not branding mood
- polish interaction states: focus, loading, empty, warning, success

## Anti-patterns to avoid

- nested cards and floating cards for everything
- purple gradients, glow-heavy chrome, or novelty AI motifs
- gray text on tinted panels with weak contrast
- ornamental pills, circles, and status chips everywhere
- fake analytics tiles without decisions attached
- bounce or playful motion in serious operator UI

## Good fits

- AI runtime consoles
- security and audit tools
- evaluation dashboards
- multi-agent orchestration surfaces
