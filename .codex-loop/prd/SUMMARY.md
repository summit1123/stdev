# Project Summary

프로젝트명: Diary to Discovery

골든 패스:

1. 사용자가 일기 이미지를 올린다.
2. 시스템이 텍스트를 추출하고 수정 단계를 보여준다.
3. 과학 해석, 질문 씨앗, 게임형 콘텐츠, 상황 이미지를 생성한다.
4. 시나리오와 음성을 바탕으로 12초 과학 해석 영상을 만든다.
5. 결과를 저장하고 미션 로그와 함께 다시 본다.

데모에서 반드시 보여줄 것:

- 실제 이미지 업로드
- 10~15초 내 구조화 결과 표시
- 상황 이미지와 과학 해석 결과
- 음성 재생
- 실제 12초 영상 재생
- 우측 inspector에서 상태와 최근 세션 다시 보기
- 샘플 3종 fallback
